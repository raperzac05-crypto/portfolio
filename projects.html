Photon Laser Tag System

Developed a multiplayer laser tag system using Python, Pygame, and PostgreSQL. The application manages player entry, team assignment, and real-time gameplay visualization.

Implemented UDP socket communication to handle live game events, including player hits, friendly fire, and base interactions. Designed the system to process incoming network data and update the display dynamically during gameplay.

Integrated a PostgreSQL database to store player information and support automatic codename assignment and team balancing.

Technologies Python, Pygame, PostgreSQL, UDP Networking, Git
